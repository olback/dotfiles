# dotfiles

Install:
```terminal
git clone git@github.com/olback/dotfiles $HOME/.dotfiles
.dotfiles/install.sh
```

Install ZSH plugins:
```terminal
.dotfiles/install_zsh_plugins.sh $ZSH_CUSTOM
```
